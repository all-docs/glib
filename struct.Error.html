Struct

GLib Error

Description [src] 

struct GError {
  GQuark domain;
  gint code;
  gchar* message;
}

The GError structure contains information about an error that has occurred.

Structure members
domain

Error domain, e.g. G_FILE_ERROR.

code

Error code, e.g. G_FILE_ERROR_NOENT.

message

Human-readable informative error message.
