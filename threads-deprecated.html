Deprecated Thread API

These APIs are deprecated. You should not use them in new code. This section remains only to assist with understanding code that was written to use these APIs at some point in the past.

Deprecated thread creation/configuration functions:

  • GThreadPriority
  • GThreadFunctions
  • g_thread_init()
  • g_thread_get_initialized()
  • g_thread_set_priority()
  • g_thread_foreach()
  • g_thread_create()
  • g_thread_create_full()

Deprecated static variants of locking primitives:

  • GStaticMutex
  • GStaticRecMutex
  • GStaticRWLock
  • GStaticPrivate

Deprecated dynamic allocation of locking primitives:

  • g_private_new()
  • g_mutex_new()
  • g_mutex_free()
  • g_cond_new()
  • g_cond_free()
  • g_cond_timed_wait()
