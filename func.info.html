A convenience function/macro to log an informational message.

Seldom used.

If g_log_default_handler() is used as the log handler function, a new-line character will automatically be appended to @â€¦, and need not be entered manually.

Such messages are suppressed by the g_log_default_handler() and g_log_writer_default() unless the G_MESSAGES_DEBUG or DEBUG_INVOCATION environment variables are set appropriately. If you need to set the allowed domains at runtime, use g_log_writer_default_set_debug_domains().

If structured logging is enabled, this will use g_log_structured(); otherwise it will use g_log(). See Using Structured Logging.
