GLibMemVTable

Description [src] 

struct GMemVTable {
  gpointer (* malloc) (
    gsize n_bytes
  );
  gpointer (* realloc) (
    gpointer mem,
    gsize n_bytes
  );
  void (* free) (
    gpointer mem
  );
  gpointer (* calloc) (
    gsize n_blocks,
    gsize n_block_bytes
  );
  gpointer (* try_malloc) (
    gsize n_bytes
  );
  gpointer (* try_realloc) (
    gpointer mem,
    gsize n_bytes
  );
}

A set of functions used to perform memory allocation. The same GMemVTable must be used for all allocations in the same program; a call to g_mem_set_vtable(), if it exists, should be prior to any use of GLib.

This functions related to this has been deprecated in 2.46, and no longer work.
