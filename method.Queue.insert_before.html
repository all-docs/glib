Method

GLib Queue insert_before

since: 2.4

Declaration [src] 

void
g_queue_insert_before (
  GQueue* queue,
  GList* sibling,
  gpointer data
)

Description [src]

Inserts data into queue before sibling.

sibling must be part of queue. Since GLib 2.44 a NULL sibling pushes the data at the tail of the queue.

Available since: 2.4

This method is not directly available to language bindings.

Parameters

sibling

Type: A list of gpointer

A GList link that must be part of queue, or NULL to push at the tail of the queue.

The argument can be NULL.
The data is owned by the caller of the method.
data

Type: gpointer

The data to insert.

The argument can be NULL.
The data is owned by the caller of the method.
