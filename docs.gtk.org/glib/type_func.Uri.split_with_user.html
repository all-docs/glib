Description [src]

Parses uri_ref (which can be an absolute or relative URI) according to flags, and returns the pieces. Any component that doesnâ€™t appear in uri_ref will be returned as NULL (but note that all URIs always have a path component, though it may be the empty string).

See g_uri_split(), and the definition of GUriFlags, for more information on the effect of flags. Note that password will only be parsed out if flags contains G_URI_FLAGS_HAS_PASSWORD, and auth_params will only be parsed out if flags contains G_URI_FLAGS_HAS_AUTH_PARAMS.
