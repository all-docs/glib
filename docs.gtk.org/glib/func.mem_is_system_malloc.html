Checks whether the allocator used by g_malloc() is the systemâ€™s malloc implementation. If it returns TRUE memory allocated with malloc() can be used interchangeably with memory allocated using g_malloc(). This function is useful for avoiding an extra copy of allocated memory returned by a non-GLib-based API.

Deprecated since: 2.46

GLib always uses the system malloc, so this function always returns TRUE.

Return value

Type: gboolean

If TRUE, malloc() and g_malloc() can be mixed.
